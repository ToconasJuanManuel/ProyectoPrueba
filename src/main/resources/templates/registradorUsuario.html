REGISTRO DE TRIPULANTES

INSTRUCCIONES

Ha ingresado en el formulario de carga de tripulantes.

Llenado el formulario podra realizar las siguientes

CARGAR TRIPULANTE

Tras llenar el formulario, podra cargar un tripulante usnado cualquiera de los botones pero el boton REGISTRAR TRIPULANTE limpiara los campos para poder ingresar otro tripulante mientras FINALIZAR REGISTRO ingresara el nuevo tripulante y finalizara la carga

FINALIZAR REGISTRO

Si el registro del vehiculo tiene al menos un tripulante podra usar este boton para finalizar la carga de tripulantes, si el formulario esta incompleto el nuevo tripulante no se cargara caso contrario se ingresara el nuevo tripualante y se finalizara el registro
